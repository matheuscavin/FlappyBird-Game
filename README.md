# FlappyBird-Game

This is a basic FlappyBird game using Pygame.

## Requeriments
- Python 3.7.8
- Libraries: Pygame, Time, Raandom

## Clone
Clone this repository to your local machine using:
```shell
$ git clone https://github.com/matheuscavin/FlappyBird-Game.git
```
